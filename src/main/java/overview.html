The Lord of the Rings: Duel for Middle-earth

This project is an implementation of the Board game The Lord of the Rings: Duel for Middle-earth.

Features

  • A model implementation of the game
  • A console interface to play the game

Quickstart

  1. Prerequisites
    • Java 22 SDK installed
    • Maven 3.8+ installed
  2. Build the project 
    
mvn clean install
  3. Run the console app

    Use Maven’s exec plugin (already configured in the POM):

    
mvn exec:java@play
  4. Sample gameplay 
    
After running the program, you will be able to manipulate the state with console commands.
To list the possible commands, use the `list` command. To quit the application, use the `quit`
command.
